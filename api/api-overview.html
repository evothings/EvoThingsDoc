API Overview

The Evothings Viewer app ships with selected Cordova plugins, which are listed below.

Evothings also provides a number of JavaScript libraries for IoT, built on top of the Cordova plugins.

Below you find API documentation for plugins and libraries.

Evothings JavaScript libraries

Evothings JavaScript libraries provide essential functionality for IoT enabled mobile apps:

  • evothings - Functions for asynchronous loading of JavaScript files and platform checking functions.
  • evothings.easyble - High-level functions for accessing BLE devices. This library is written on top of the BLE plugin, and asynchronously includes the evothings.util library.
  • evothings.eddystone - Library for scanning for Eddystone devices. This library is written on top of the BLE plugin, and asynchronously includes the evothings.easyble library.
  • evothings.util - Functions for parsing numbers encoded as binary data. This library is used by the EasyBLE library.
  • evothings.tisensortag High-level library for the TI Bluetooth Smart SensorTag. This library asynchronously includes the EasyBLE library.
  • Microcontrollers - Basic libraries for making it easier to communicate with microcontroller, such as Arduino, Nordic Semiconductor, and RFduino.
  • evothings.iotsensor - High-level functions for Dialog Semiconductor's IoT Sensor. This library asynchronously includes the evothings.easyble and evothings.util library.

Include these libraries using a script tag in the HTML code, or load them asynchronously using the evothings.loadScript() function.

When using a library that asynchronously includes another library using evothings.loadScript() it is good practice to always wait for all libraries to have loaded before calling any library functions. This is typically done in the Cordova 'deviceready' event handler. Here is an example:

function initialise()
{
    // All async scripts are loaded, the app is
    // ready to be initialised.
}

document.addEventListener(
    'deviceready',
    function() { evothings.scriptsLoaded(initialise) })

Browse the JSDoc generated API documentation for the Evothings JavaScript libraries to learn more.

The source code for the libraries is found on GitHub: https://github.com/evothings/evothings-libraries

Some of the JavaScript libraries are written to communicate with Arduino sketches. The Arduino source code is found here: https://github.com/evothings/evothings-libraries/tree/master/arduino
